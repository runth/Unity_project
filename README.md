# 게임 실행 전 디스플레이 설정
게임을 원활하게 실행하기 위해, 다음 단계를 따라 디스플레이 설정을 조정해주세요:

1. **디스플레이 설정 열기**

  - **Windows**에서 시작 메뉴를 열고, **설정**을 선택합니다.
  - **시스템**을 선택한 후, **디스플레이**로 이동합니다.

2. **해상도 조정**

  - **디스플레이 해상도** 섹션에서 **1920 x 1080** 해상도를 선택합니다.

3. **설정 적용**

  - 해상도 선택 후, 변경 사항을 적용합니다.


이제 게임을 실행해도 좋습니다!
